# FACE: Counterfactual Pairwise Evaluation of Physiognomic Social Bias in Vision-Language Models

This repository contains the official implementation of FACE, a benchmark designed to systematically quantify physiognomic social bias in Vision-Language Models (VLMs). FACE disentangles visual attributes from other confounds to measure how multimodal systems map appearance to social character.
